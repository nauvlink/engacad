HTML (Hypertext Markup Language) is the code that is used to structure a web page and its content.

CSS (Cascading Style Sheets) is the language used for the presentation of Web pages, including colors, formats, and fonts.

Github is a coding platform where people can collaborate on projects and host websites.

Setting up Github.

Phase 1: Setting up a Github Account.
First, go to Github.com and sign up for an account using ur email.

Phase 2: Create a Github Repository.
Create a new Github repository.
Modify the Repository name preferably based on your website name.
Make the repository public for friends and teachers to view.
Adding a description can be done.

Phase 3: Host a Github Website.
Firstly, Upload your website's HTML and CSS files into the repository.
Go to Repository settings after finishing creating the repository.
Scroll down all the way until Github Pages appear.
Select Main as the branch and Save.
A pop-up will appear showing the Website URL.

Congratulations. You have completed Part 1.
